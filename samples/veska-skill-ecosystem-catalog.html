/plugin marketplace add anthropics/skills
/plugin install document-skills@anthropic-agent-skills

Community flagship · MIT · needs your go-ahead

UI/UX Pro Max Authorize

The most popular community design skill (~88k★): 67 UI styles, 161 palettes, 57 font pairings, 99 UX guidelines, 25 chart types, searchable via bundled Python (inspected: pure-stdlib, no network/exec). Ships executable scripts → explicit owner authorization required to vendor. Or run its installer yourself:

npm i -g uipro-cli && uipro init --ai claude
