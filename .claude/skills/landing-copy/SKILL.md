# Landing-page copy

Words that make a premium site convert. Clarity beats cleverness every time.

## Principles
- **One job per section.** Each block answers one question the visitor has next.
- **Benefit-led headline, specific proof.** Say the outcome, then back it with a
  concrete number, name, or fact — never vague hype.
- **One primary CTA**, repeated. Secondary actions stay visually quieter.
- **Speak to the buyer**, not about yourself. "You/your" > "we/our".
- **Cut filler.** If a word can go, it goes.

## Section-by-section
1. **Hero** — eyebrow (who it's for) · headline (the outcome) · 1-line subhead
   (how) · primary CTA · 1 proof chip (rating, logo count, guarantee).
2. **Social proof** — logos, a stat, or one sharp testimonial.
3. **Features → benefits** — lead each with the benefit; the feature is the
   reason-to-believe underneath.
4. **Objection handling** — name the top 2–3 hesitations and answer them
   (price, switching effort, risk) — often as an FAQ.
5. **Pricing** — plain, scannable, one recommended option.
6. **Final CTA** — restate the core promise + the single action.

## Voice
Pull tone from the **brand-kit**. Match the sector: a law firm reads measured;
a fitness studio reads punchy. Keep it human.

## Guardrails
- **Honesty.** Never fabricate testimonials, named clients, stats, or awards.
  Use placeholders the client fills with real proof, and label them as such.
- Avoid superlatives you can't substantiate ("best", "#1") unless true and
  cited.
