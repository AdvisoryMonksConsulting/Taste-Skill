# Advisory Monks — Brand Kit

Everyday brand assets for Advisory Monks Consulting.

## Contents
- **Advisory-Monks-Brand-Guide.pdf** — the guidelines (logo, colour, type, voice, applications).
- **assets/** — logo.svg (on light), logo-rev.svg (on navy), logo-white.png (raster, for email/social).
- **templates/** — ready-to-use, open in a browser:
  - `email-signature.html` — open, select-all, copy, paste into Gmail/Outlook signature settings. Edit name/title in the file first.
  - `letterhead.html` — print to PDF (A4) for documents/proposals.
  - `social-post.html` — 1200×1200 LinkedIn/Instagram template; edit the headline, screenshot or print to image.
  - `business-card.html` — front + back at print size; send to a printer or export to PDF.
